# EventPro

EventPro is a modern, frontend-focused event management demo application built to showcase clean UI design, intuitive user experience, and scalable project structure. It is intended as a demo and presentation-ready project rather than a fully functional production system.

---

## 🌐 Live Demo

After deployment via GitHub Pages, the demo will be available at:

https://<your-github-username>.github.io/EventPro/

---

## 🛠 Project Overview

EventPro demonstrates how an event organization platform can look and feel from a user interface perspective. It includes structured pages, reusable UI components, and responsive layouts suitable for showcasing event listings, dashboards, and user flows.

This project does not include backend functionality or database integration. It is designed purely for demonstration, learning, and portfolio use.

---

## 🚀 Tech Stack

This project is built using modern frontend technologies:

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
